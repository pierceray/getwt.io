with homebrew

``` shell
brew install wellington
```

with go

``` shell
go get github.com/wellington/wellington/wt
```

use prebuilt binaries on the github [release page](https://github.com/wellington/wellington/releases)

* * *
Run wellington

``` shell
wt -v
```
